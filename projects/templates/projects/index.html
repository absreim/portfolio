Simmer

Save recipes from popular recipe sites for sharing and offline viewing. Get recommendations from users with similar tastes.

To save a recipe, navigate to the recipe page of any of 5 supported sites in Google Chrome and use the Chrome extension. Alternatively, one can enter the URL into the web application.

For offline viewing, one can save the web application as a desktop or mobile app. Previously visited pages are automatically cached.
